iOS (V2Box / Streisand)

Скопируй VLESS-ссылку → импортируй в приложение.

  • Нажми “Скопировать конфиг”
  • Открой V2Box → Import → From Clipboard
  • Подключи профиль

Android (v2rayNG)

Импорт из буфера — самый быстрый вариант.

  • Нажми “Скопировать конфиг”
  • v2rayNG → + → Import from Clipboard
  • Подключить

Desktop (Windows / macOS)

Подойдёт v2rayN / Nekoray / Hiddify.

  • Скопируй конфиг
  • Импорт по ссылке или из буфера
  • Подключи профиль
